network settings:

1.device settings
/etc/sysconfig/network-scripts
BOOTPROTO=none|static|dhcp
IPADDR=192.168.1.254
GATEWAY=192.168.1.1
NETMASK=255.255.255.0

2.device up|down
ifconfig eth0 up|down
ifup|ifdonw eth0

3.network service
/etc/rc.d/network start|restart|stop

4.route
route add -net 192.168.0.1 netmask 255.255.255.0 gw 192.168.0.1 metric 1 dev eth0
route add default gw 192.168.0.1 eth0
route del default eth0
*default aka 0.0.0.0
*gw must avaliable; del may failed if parameters not match when add; eth0 aka dev eth0;

5.multiple devices route
WLAN need gateway,LAN no need: 
    Destination   Gateway        Genmask   Use Iface
    default       192.168.1.1    0.0.0.0   eth0
better use eth0 as router then no need manual route setting;

6.trouble shoot
some router reuse WLAN/LAN, need select and set ports or some LAN doesn't work;
